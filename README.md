## 🚀 **GETTING STARTED**

### 🎯 **Quick Launch**
```bash
# Clone the kingdom
git clone https://github.com/yourusername/mufasa-ai.git

# Enter the pride lands  
cd mufasa-ai

# Install dependencies
npm install

# Start the 3D experience
npm run dev
```
